## 功能实现

- [x] 短信登陆
- [ ] 账号密码登陆
- [x] 主页视频推荐
- [x] 视频搜索
- [x] 评论区
- [x] 评论区楼中楼
- [x] 相关视频
- [x] 热搜
- [x] 视频播放
- [x] 弹幕
- [ ] 直播
- [ ] 视频分区
- [ ] 动态
- [ ] 用户投稿
